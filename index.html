Una caracteristica que define a un sistema complejo es la cantidad y calidad de las interacciones que suceden entre estas mismas

Una interaccion fuerte se puede definir con una linea gruesa

Una interaccion debil se puede definir con una linea delgada

Una interaccion intermitente se puede definir con una linea punteada

Una interaccion da lugar a una propiedad emergente que puede autoregular un sistema
